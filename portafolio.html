Portafolio Profesional

Proyectos desarrollados aplicando conceptos de diseño, estructura y programación web moderna.

Resumen

Estos proyectos reflejan mi proceso de aprendizaje y aplicación práctica en el desarrollo web, combinando herramientas como HTML, CSS y Bootstrap. Cada uno representa un avance en el dominio de estructura, diseño y funcionalidad digital, demostrando la integración entre técnica y estética.

Proyectos Realizados

PROYECTO CAPTURA DEL PROYECTO
Desafío 1 – Curriculum
Creación de una estructura HTML semántica para construir un currículum digital claro y ordenado.
Desafío 2 – Landing Page CSS
Diseño de una landing page moderna aplicando estilos CSS, tipografías y layout adaptable.
Desafío 3 – Iguana Page
Uso de Flexbox y posicionamiento para recrear una página temática con estilo profesional y responsivo.
Desafío 4 – Cuppon
Implementación de una interfaz de e-commerce utilizando Bootstrap y componentes dinámicos.
Desafío 5 – Olivia Ros
Portafolio completo que combina Bootstrap, diseño responsive y estilo visual moderno para presentar proyectos fotográficos.
